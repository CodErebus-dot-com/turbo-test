Quick start after installation:

```

```

## Tests

Run test scenarios in `tests/**/*.spec.ts`:

```

yarn test

```

Run test scenarios and show code coverage:

```

yarn test:coverage

```

Run test scenarios and prepare report:

```

yarn test:report

```
